# aws-boto-functions
Collection of aws-boto3 functions that will make a DevOps Engineer's life easy.

<br>

## List of AWS Functions
#### RDS
* [List RDS Instances](scripts/list_rds_instances.py) - Obtain list of all the RDS instances for every given region for every given AWS account
* [List RDS Snapshots](scripts/list_rds_snapshots.py) - Obtain list of all the RDS Snapshots for every given region for every given AWS account
